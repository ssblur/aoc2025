# Advent of Code 2025

*By Pat Hallbick*

## Implementation Details
Written in Go as a portfolio piece.

## How to run
Data is located in day roots as input.txt, modules are meant to be run from project root.

(e.g. for Day 1 Part 1, data is in `day1/input.txt` 
and the project is run from root with `go run ./day1/part1`)
